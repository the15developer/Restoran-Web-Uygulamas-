# Restoran-Web-Uygulaması

HTML, CSS, JavaScript, Bootstrap, Node.js + Express.js ve MongoDB teknolojileri ile bir restoran web uygulaması geliştirdim. Güncel hali Github hesabımdan takip edilebilir.

I developed a restaurant web application with HTML, CSS, JavaScript, Bootstrap, Node.js + Express.js and MongoDB technologies. The current version can be followed on my Github account. The application implements important functionalities for clients and admins, like account creation and user authentication, access based on user role, admin dashboard for managing the application, ability to leave ratings and reviews, make reservations and requests/complaints for clients and much more.
